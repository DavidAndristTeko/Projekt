SMART 10
---------------------------------------------------------------------------
Vor dem ausführen sicherstellen, dass alle Dateien sich im selben Ordner befinden

Nur die Eingaben tätigen, welche das Spiel auffordert: Enter, y oder n

Im Code werden bestimmte Ausdrücke genutzt, welche wir hier gerne spezifizieren:

Frage - Die Frage die Beantwortet wird

Option - Die Sache, zu der die Frage gerade beantwortet wird. Wenn die Frage also "Handelt es sich um ein Säugetier?" ist, wären Optionen die 10 Tiere, die abgefragt werden wie z. B. "Krokodil" oder "Wal"

Fragenblock - Ein Dict welches die Frage, die 10 dazugehörigen Optionen und deren Antworten enthält
